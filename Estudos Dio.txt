- Afinal o que é o Git? 
: É uma plataforma de hospedagem de código e arquivos com controle de versão usando o Git.
Ele permite que programadores, cadastrados na plataforma contribuam em projetos privados
e/ou Open Source de qualquer lugar do mundo.
- O que é git bash?
É um terminal de uso extendido para otimizar o uso do Git.
